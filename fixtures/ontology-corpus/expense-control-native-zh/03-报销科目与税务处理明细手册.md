## 写在前面的话（请新同事和报销经办人务必先看完这一段）

我是财务部核算与税务会计组的，平时大家在鼎财云（内部都叫"鼎财"）里提交的报销单，最后落到金蝶云做账、做税务处理的就是我们组。这份手册不是给我们财务自己看的，是写给你们——各事业部、研发中心、总部职能、还有现场外包同事看的。

为什么要写这么一大本？因为这两年集团报销出问题最多的，不是金额超了（金额超了系统会卡），而是**科目走错、发票不合规、该抵的没抵、不该报的硬报**。一笔报销走错科目，轻则被财务退单返工，重则影响集团企业所得税汇算清缴、增值税进项抵扣，甚至引来税务稽查。我们一年因为发票不合规、科目错配而做的进项转出、纳税调增，金额不小，年底老板看报表的时候是要问责到事业部的。

所以请把这份手册当成一个"查字典"的工具：你要报一笔费用，先想清楚这是**什么性质的支出**，翻到对应科目那一节，照着【适用场景】【单据发票要求】【税务处理与抵扣】【限额标准】【常见错误】这五块对一遍，基本就不会错。

几个**必须先记死**的大原则，后面每个科目都会反复用到：

1. **发票抬头一律是集团统一抬头**，不许写个人名字。集团抬头、税号、开户行、地址电话，鼎财云首页右上角"我的开票信息"里都有，自己复制给对方，别凭记忆报税号。研发中心和各事业部抬头可能不一样（有的是子公司主体），开票前一定先在鼎财里查清楚自己归属哪个开票主体。

2. **凡是想抵扣进项税的，必须拿增值税专用发票（我们简称"专票"）**，而且专票必须在系统里通过"票通"查验真伪、并在认证期内（自开票之日起一般我们要求 90 天内提交报销，逾期专票勾选抵扣的风险由部门承担）。

3. **能取得专票的尽量取专票，但有几类费用按税法本身就不让抵**，比如招待、福利、给员工的集体福利，这些哪怕你拿了专票也不能抵，反而要做进项转出，所以这几类直接开普票就行，别费劲去要专票（后面每个科目会写清楚）。

4. **电子发票（全电发票 / 数电票）和纸质发票同等效力**，现在集团强烈建议大家都收数电票，直接在鼎财里上传 PDF/OFD 原件，系统自动调"票通"查验。纸质票该贴的还得贴、该寄回财务影像中心的还得寄。

5. **金额阈值决定审批层级**，业务线和财务线两条审批是并行加串联的。业务线：经办人 → 直属上级 → 部门负责人 → 事业部分管领导（金额越大走得越高）；财务线：费用会计初审 → 财务复核 → 财务经理 / 财务总监终审（金额大、科目敏感的往上走）。每个科目下面我会写清楚具体阈值。

6. **三类员工政策不同**。内勤员工走"日常办公额度"；研发员工有"研发专项额度 / 差旅额度 / 设备额度"，研发费用归集口径更严（关系到研发费用加计扣除）；外包/驻场员工报销范围受限，且**必须经甲方项目经理或对接人审批**后我们才受理，外包同事很多科目压根不能报，后面会单独点名。

好，下面进正文。先讲单据和发票的通用规矩，再一个一个科目讲透，最后讲不同场景怎么组合走科目。

## 第一章 单据与发票的通用要求（所有科目都适用，先看这章）

### 1.1 发票类型，先分清楚

大家平时拿回来的"票"，在我们税务处理上分这么几类，性质差别很大：

| 票据类型 | 长什么样 | 能不能抵进项 | 备注 |
|---|---|---|---|
| 增值税专用发票（专票） | 票面有"增值税专用发票"字样，有税额栏，需购方完整税号、地址电话、开户行 | 能抵（除非属于税法规定不得抵扣项目） | 想抵扣就必须是这个 |
| 增值税普通发票（普票） | 票面"增值税普通发票"，有的只写购方名称税号 | 不能抵进项 | 招待、福利等本就不让抵的，开普票即可 |
| 数电票 / 全电发票 | 电子的，PDF 或 OFD，分专用、普通两种 | 看是专用还是普通 | 集团首选，直接传系统 |
| 增值税电子普通发票 | 老式电子普票 PDF | 不能抵 | 等同普票 |
| 火车票 / 飞机票行程单 | 铁路电子客票、航空运输电子客票行程单 | 国内旅客运输可按规定计算抵扣（凭注明旅客身份信息的票据） | 后面差旅费专门讲 |
| 出租车票 / 网约车行程单 | 滴滴等开的电子普票 + 行程单 | 网约车开专票可抵，纸质出租车发票不能抵 | 后面市内交通讲 |
| 过路过桥费票据 | 通行费数电票 | 可抵 | 注意区分桥闸通行费 |
| 收据 / 白条 | 没有税号、不是税务发票 | 一律不能抵，多数情况不能报 | 后面讲哪些例外可凭收据 |

**核心一句话：要抵扣，认"专票/数电专票/旅客运输票据"；招待福利类，认普票就行。**

### 1.2 发票必须满足的硬条件（缺一不可）

不管什么科目，发票要被我们受理，必须同时满足：

1. **抬头正确**：购买方是集团或所属子公司的全称，税号、地址、电话、开户行（专票需要全套，普票至少要名称和税号）。手写抬头、抬头写成"个人"或写错字的，一律退。
2. **真伪可查**：能在"票通"查验通过。假票、查无此票、重复报销的，直接进黑名单，按制度可追责。
3. **在有效期内提交**：原则上发票开具之日起 **90 天内**必须提交报销。跨年的发票，最晚到次年 **1 月 15 日**（集团年度账结账前）必须报完，否则只能走"以前年度损益调整"，要部门负责人特批，且专票可能错过抵扣期。
4. **品名规范**：发票"货物或应税劳务、服务名称"栏要写清楚买的是什么，不能只开"办公用品""材料一批"这种笼统的。开"办公用品"的，必须附销货清单（销售方系统开具的清单，不是手写的）。开"礼品""日用品"这种含糊品名的，我们一律按招待或福利从严处理，不让抵。
5. **金额、税率正确**：税率栏要对得上（现代服务业一般 6%，有形动产租赁 13%，交通运输 9%，生活服务 6%，小规模 1%/3% 等）。税率明显错的我们会退回重开。

> 踩坑提醒（真实发生过）：某事业部去年报一笔 8 万的设备款，对方开的专票品名是"货物一批"，没附清单，金税系统比对不上，被税务预警，我们紧急让供应商重开，差点错过认证期。**专票品名含糊 + 无清单 = 不予受理，别嫌麻烦。**

### 1.3 鼎财云提交的通用动作

每一张报销单的命，都从你在鼎财里点"新建报销"开始。通用步骤：

1. 选**报销类型/科目**（这是最关键的一步，选错后面全错）。
2. 选**成本中心 / 项目**（费用记到谁头上，研发的还要选研发项目号）。
3. 录**金额、发票号、税额**，上传发票原件（数电票传 PDF/OFD，纸票拍照+寄回）。
4. 系统自动调"票通"查验、查重（同一张发票号报过就会被拦）。
5. 填**事由说明**（写清楚干了啥，别写"业务需要"四个字糊弄）。
6. 提交，进业务审批线 + 财务审批线。

> 口头提醒：事由写得越清楚，退单越少。我们财务一天看几百单，你写"4月12日陪XX客户考察东莞工厂产线，午餐招待4人"，比你写"招待费"快十倍过审。

## 第二章 各报销科目逐一讲透

下面每个科目我都按【适用场景】【单据/发票要求】【税务处理与抵扣】【限额标准】【常见错误与不可报情形】【举例】六块来写。限额数字都是集团 2026 年现行标准，以鼎财系统里的最新配置为准，有调整以系统为准。

---

### 2.1 差旅费

差旅费是报销量最大、最容易出错的一类，所以拆开讲。差旅费在金蝶里挂"管理费用—差旅费"（研发项目相关的差旅，按规定归集到"研发费用—差旅"，后面研发那节讲）。差旅费包含：机票/火车票、市内交通（出差期间的）、住宿、出差补贴四块。

**前置规矩：出差必须先有出差申请。** 不管什么员工，出差前要在鼎财提"出差申请单"，写明出差事由、起止城市、起止日期、预计费用。**没有事先批准的出差申请，回来报销一律不受理**（紧急出差可事后补，但要直属上级+部门负责人双签说明原因，且 3 个工作日内补单）。预计总费用达到 5,000 元的，按"事前审批"走预申请（外包员工无论金额都要事前审批）。

#### 2.1.1 机票 / 火车票（城际交通）

**适用场景**：出差往返、跨城市公务行程的大交通。

**单据/发票要求**：
- 飞机票：要"航空运输电子客票行程单"（就是大家说的行程单），上面有旅客姓名、票价、机建燃油、保险。光有值机牌、订单截图不行，必须是行程单。
- 火车票/高铁票：现在都是电子客票，报"铁路电子客票"（12306 开的电子发票/报销凭证），上面有旅客身份信息、票价。纸质蓝色报销凭证也认。
- 改签费、退票手续费：要单独的票据或行程单注明，能报但要写清原因。

**税务处理与抵扣**：
- 国内旅客运输可以抵扣进项！飞机票按行程单上"票价+燃油附加费"（不含机建）×9% 计算抵扣（注明旅客身份信息的电子客票行程单）；火车票按票面金额÷(1+9%)×9% 抵扣。
- **重要**：能抵的前提是**国内**、**本单位员工**、**注明旅客身份信息**。国际机票不能抵。给非员工（比如面试候选人、外部专家）买的票，不走差旅费抵扣，走相应科目（招聘费/会议费等）且不抵。
- 我们核算组会在做账时按上面口径自动算抵扣，你报的时候按票面全额（含税）填报销金额即可，不用自己拆。

**限额标准**（按员工职级，集团现行）：
- 飞机舱位：副总裁及以上可公务舱；总监/部门负责人及研发首席专家级经济舱（特殊长途经分管领导批可公务舱）；其余员工一律经济舱。**超标部分自理**，系统会按职级卡舱位价格区间，明显超标的会预警退单。
- 高铁：总监及以上可一等座/商务座（4 小时以上行程经批可商务座）；其余员工二等座。
- 提倡提前预订，无特殊原因临行前 24 小时内高价订票的，超出合理价部分可能被要求说明。

**常见错误与不可报情形**：
- 用别人的票报销（票上姓名不是本人，除非是同行公务陪同且事由写清，否则退）。
- 只传订单截图、付款截图，没有行程单/电子客票——不受理。
- 把家属、私人同行的票一起报——这是红线，查实按违规处理。
- 行程与出差申请的城市、日期对不上——退单核对。

**举例**：研发中心王工去深圳出差 3 天，往返高铁二等座共 1,090 元，报"差旅费—城际交通"，传 12306 电子发票两张，事由写"4月15-17日赴深圳供应商XX做样机联调"。核算组做账时按 1090÷1.09×9%=90 元算进项抵扣。

#### 2.1.2 住宿费

**适用场景**：出差期间的酒店住宿。

**单据/发票要求**：
- 必须酒店开的增值税发票，写明集团抬头税号。**想抵扣的开专票**（住宿属于可抵扣的现代服务，6%）。
- 注明入住人姓名、入住离店日期最好，方便核对天数。
- 平台（携程等）代订的，要平台开的发票或酒店直接开的发票，注意别拿成"代订服务费"的票冒充住宿费。

**税务处理与抵扣**：
- 住宿费取得专票**可以抵扣**进项税（6%）。所以出差住宿尽量要专票！这是少数几个鼓励大家费劲也要专票的科目。
- 出差期间的住宿是抵扣的；但如果是公司请客户、请外部人员住宿，那性质变成招待，不能抵（见招待费）。

**限额标准**（每人每晚，按城市分档，集团现行）：
- 一线城市及省会（北上广深、杭州、成都等）：普通员工 ≤ 450 元/晚，总监及以上 ≤ 700 元/晚。
- 二线城市：普通员工 ≤ 350 元/晚，总监及以上 ≤ 550 元/晚。
- 其他城市：普通员工 ≤ 280 元/晚，总监及以上 ≤ 450 元/晚。
- 超标部分自理。同性别同事同行出差提倡合住（合住的可按人头算）。
- 现场长期驻场的外包/项目人员，住宿一般由项目统一安排或包干，不走个人住宿报销（见 2.16 外包）。

**常见错误与不可报情形**：
- 把客房里的迷你吧、洗衣、付费电影一起开进住宿发票报销——这部分要剔除，属个人消费。
- 出差还没结束就报、或住宿日期超出出差申请天数——退。
- 拿"住宿费"发票实际是民宿/无资质场所开的非正规票——查验不过不受理。

**举例**：内勤员工小李去上海出差 2 晚，某酒店每晚 420 元，开住宿专票 840 元，报"差旅费—住宿"。核算组按 840÷1.06×6%=47.55 元抵扣。若小李住了 500 元/晚（超一线 450 标准），超出的 50×2=100 元自理，系统会提示。

#### 2.1.3 市内交通费（出差期间的）

**适用场景**：出差在外地，往返机场/车站、办事地点之间的市内打车、地铁、公交。

**单据/发票要求**：
- 网约车（滴滴等）：电子发票+行程单，行程单上有时间、起止地点。
- 出租车：纸质发票（注意纸质出租车票不能抵进项）。
- 公交地铁：有票尽量留，金额小的可据实凭行程记录报。

**税务处理与抵扣**：
- 网约车取得专票的可以抵（9% 或 6%，看开票方）；纸质出租车发票不能抵。
- 这块抵扣金额一般不大，重点是据实、合理。

**限额标准**：出差市内交通据实报销，但单日市内交通明显异常（比如一天打车 500 元以上）要写明原因。出差期间不再另发市内交通补贴。

**常见错误**：把私人行程的打车混进去；行程单时间在深夜且与公务无关的要解释。

> 注意：**本地日常上下班的市内交通不算差旅费**，本地通勤打车走 2.3"市内交通费"那一节，规则完全不同，别混。

#### 2.1.4 出差补贴（伙食与杂费补贴）

**适用场景**：出差期间的伙食、市内零星杂费，集团按天给补贴，**不需要发票**，按出差天数据实计算。

**单据/发票要求**：**无需发票**，凭批准的出差申请单和实际出差天数，系统自动按标准计算。这是少数不要票的项目，但正因为不要票，查得严，天数务必真实。

**税务处理**：出差补贴在标准内的，属于差旅费性质，企业所得税可税前扣除，**不并入个人工资缴个税**（这是合规出差补贴的好处，前提是标准合理、有据可查）。超标准发放的部分有被认定为工资、需补个税的风险，所以别乱抬高。

**限额标准**（每人每天，集团现行）：
- 一线城市：120 元/天（含伙食+市内零星）。
- 二线城市：100 元/天。
- 其他地区：80 元/天。
- 出差当天往返（未过夜）按半天算。出差天数以出差申请批准并实际发生为准。
- **如果出差期间伙食由对方/项目统一安排（管饭），补贴相应扣减。**
- 注意：拿了补贴就不能再单独报出差期间的个人餐费发票，二选一，重复报销是违规。

**常见错误与不可报情形**：
- 既领补贴又报餐费发票——典型重复报销，查实退还并通报。
- 虚报出差天数（人没去/提前回来还按全程报）——红线，按舞弊处理。
- 把出差补贴和招待费混——补贴是自己的伙食，招待是请客户吃饭，完全两回事。

**举例**：内勤员工赵姐去成都（二线，100元/天）出差 4 天，补贴 400 元，无需发票，系统按出差申请天数自动核算，不计个税。

### 2.2 业务招待费

这是**全集团审批最严、税务限制最死**的科目，请重点看。

**适用场景**：因业务需要宴请客户、供应商、合作方，或商务考察中招待对方产生的餐饮、必要的礼品。注意——招待的对象是**外部人**，请自己同事吃饭那不叫招待（那要看是不是团建或福利）。

**单据/发票要求**：
- 餐饮发票（增值税普通发票即可，**不要去要专票**，因为餐饮招待本就不能抵进项，要了专票还得转出，白费劲）。
- 必须在鼎财事由里写清楚：**招待事由、招待对象单位及人数、陪同我方人数、日期地点**。光一张餐费发票不写对象的，一律退。
- 大额招待（单次≥3,000元）建议附上当时的来访/接待安排或邮件佐证。

**税务处理与抵扣**：
- **业务招待费进项税一律不得抵扣**（哪怕拿了专票也不行，要做进项转出）。所以开普票即可。
- 企业所得税前扣除有**双重限制**：业务招待费按发生额的 **60%** 扣除，且**最高不超过当年销售（营业）收入的 5‰**。两个取小的。这意味着招待费花得越多，能税前扣的比例越被卡死，超出部分要纳税调增。**所以集团从严控制招待费，不是抠门，是税法卡死了。**

**限额标准**（集团现行，按人均和审批）：
- 人均餐标：陪同客户用餐人均 ≤ 200 元（一线城市重要客户经分管领导批可 ≤ 300 元）。明显超人均的要说明。
- 单次招待总额：< 1,000 元，部门负责人审批即可；1,000–5,000 元，需事业部分管领导审批；≥ 5,000 元，需分管领导+财务总监审批，且必须事前在鼎财提"招待预申请"。
- 招待用酒水：高档酒水（单瓶超 500 元）原则上不予报销，特殊重要接待经分管领导事前书面批准方可。
- 礼品类招待：单次礼品 ≤ 500 元/对象，且要写明赠送对象。烟、贵重礼品不报。

**常见错误与不可报情形**：
- 把同事聚餐、部门会餐当招待报——这是最常见的错。请外部人才算招待，内部吃饭走团建（且团建有自己的规则和频次限制）。
- 招待发票要专票去抵扣——错，不能抵，反而暴露问题。
- 事由不写招待对象——退单率最高的就是这个。
- 把招待费拆成多张小票规避审批阈值（化整为零）——查实按违规，审批权限按累计金额算。
- 春节、中秋等节点给客户送礼以"招待"名义大额报销——要走专项审批，且不能涉及不正当利益输送，合规红线。

**举例**：事业部销售经理老陈宴请某客户采购团队 4 人，我方陪同 2 人，共 6 人，餐费 1,150 元，人均约 192 元（未超 200），开餐饮普票。事由写"5月8日宴请XX公司采购李总等4人洽谈年度框架协议，我方2人陪同"。因总额 1,150 元（1,000–5,000 区间），需走到分管领导审批。做账记"管理费用—业务招待费"，进项不抵，年底汇算按 60% 和收入 5‰ 孰低做纳税调整。

### 2.3 市内交通费（本地日常）

**适用场景**：本地（常驻工作城市内）因公外出的打车、网约车、公交地铁。比如内勤去本地客户那开会、研发去本地供应商现场、加班晚归打车（符合加班政策的）。**注意和差旅期间的市内交通分开**，这是本地的。

**单据/发票要求**：
- 网约车电子发票+行程单（行程单证明时间地点和公务关联）。
- 公交地铁有票留票，无票的小额可凭行程记录。
- 出租车纸质发票。

**税务处理与抵扣**：
- 网约车专票可抵进项；纸质出租车票、公交地铁不抵（公交地铁多为普票或无票）。
- 本地市内交通整体抵扣金额小，重点是据实、和公务挂钩。

**限额标准**（集团现行）：
- 内勤员工：本地因公交通据实报销，单月本地交通费一般控制在 800 元以内，超出需部门负责人说明审批。
- 加班打车：工作日 21:00 之后或周末加班（需有加班记录），可报回家打车费，单次 ≤ 100 元（远郊经批可放宽），需备注加班事由。
- 研发员工：去本地实验/测试现场的交通据实，挂研发项目的归集到研发费用差旅。
- **外包员工的本地交通原则上不报**，确需的由项目包干或甲方对接人特批。

**常见错误与不可报情形**：
- 上下班正常通勤打车当因公报——不行，通勤不报（加班除外）。
- 周末私人出行的网约车混报——退，且影响信用。
- 行程单时间地点和填的事由对不上。

**举例**：内勤小周下班后 21:30 加班结束打车回家 45 元，有当天加班记录，报"市内交通费—加班交通"，备注"4月20日加班至21:30处理月结"，部门内审批即可。

### 2.4 办公用品费

**适用场景**：日常办公消耗品，如纸张、笔、文件夹、打印耗材、硒鼓、小额办公小电器（计算器、订书机）、桌面用品等。注意——单价较高、能用很久的设备（电脑、显示器、打印机本身）不走这里，走 2.5 设备采购。

**单据/发票要求**：
- 增值税专用发票（办公用品可抵进项，要专票！13% 或对方小规模 1%/3%）。
- **品名必须明细**：开"办公用品"必须附正规销货清单，列明具体买了什么。只开"办公用品一批"无清单的，不予受理（前面 1.2 强调过，这里再强调一遍，办公用品是无清单退单重灾区）。

**税务处理与抵扣**：
- 办公用品取得专票**可以抵扣**进项。所以请尽量要专票。
- 用于一般办公的可抵；如果买的"办公用品"其实是发给员工个人的福利（比如人手一个保温杯当节日礼物），那性质是福利，不能抵，要转出，别藏在办公用品里。

**限额标准**：
- 部门日常办公用品按部门预算包干，单次采购 < 2,000 元部门内审批；≥ 2,000 元需走部门负责人 + 行政归口审批；大批量集中采购（≥ 5,000 元）建议走行政统一采购，别个人零买。
- 个人零星办公用品垫付，单次一般 ≤ 500 元。

**常见错误与不可报情形**：
- 把私人物品（个人用的数码配件、零食、生活用品）混进办公用品报——查实退。
- 开"办公用品"无清单。
- 拿超市购物小票（非增值税发票）报——不行，超市要开正规发票。
- 把应该资本化的设备（如 5,000 元的笔记本）拆成"办公用品"报销规避固定资产管理——错，该入固定资产/低值易耗的走 2.5。

**举例**：行政小吴采购办公文具一批，A4纸、笔、文件夹共 680 元，供应商开专票并附销货清单，报"管理费用—办公费"，进项 680÷1.13×13% 抵扣。

### 2.5 设备采购

**适用场景**：采购电脑、显示器、打印机、服务器、仪器仪表、家具等**单价较高、使用期较长**的物品。研发用的专用测试设备、仪器也可能走这里或研发设备额度，见 2.6。

**单据/发票要求**：
- 增值税专用发票（设备可抵进项，13%；如果是不动产或特定项有差异）。
- 大额设备采购通常对应**采购合同/订单**和**供应商**，报销/付款时要关联合同号；走对公付款的，多数不是个人垫付报销，而是供应商挂账付款（见后文付款流程提示）。
- 验收单：达到固定资产标准的设备，入账前要有验收记录。

**税务处理与抵扣**：
- 设备（动产）取得专票**可以抵扣**进项。
- **资本化 vs 费用化**：单价达到集团固定资产标准（集团现行：单价 ≥ 5,000 元 且使用年限 > 1 年）的，要入**固定资产**，按年限折旧，不能一次性全进当期费用（除非符合税法一次性扣除政策，由财务判断）。单价 < 5,000 元的小额设备可走"低值易耗品"或一次性费用化。**所以你别把一台 8,000 元的设备硬拆成几张小票想费用化，财务一看金额和品名就会按固定资产处理。**

**限额标准 / 审批**：
- 设备采购金额大，审批升级最快：< 5,000 元 部门负责人审批；5,000–50,000 元 需事业部分管领导 + 财务复核；≥ 50,000 元 需分管领导 + 财务总监，且必须事前预申请，走采购流程比价。
- 研发专用设备走研发设备额度，由研发中心负责人 + 财务把关，关系到研发费用归集，见 2.6。

**常见错误与不可报情形**：
- 拆票规避固定资产标准——红线。
- 个人先买了高价设备再来报，不走采购流程、无比价、无合同——大额设备这样做会被退，要求补流程。
- 设备发票品名笼统（"设备一批"无型号无清单）——退。

**举例**：研发中心采购示波器一台，含税 23,000 元，供应商开专票，关联采购合同，验收后入固定资产，进项 23000÷1.13×13%≈2,646 元抵扣，设备按年限折旧。审批走到分管领导+财务复核。

### 2.6 研发费用（研发材料 / 测试 / 样机）

这是研发中心同事最关心、也是集团**最看重归集口径**的一类，因为它直接关系**研发费用加计扣除**（国家给的税收优惠，归集对了能少交不少企业所得税）。所以研发费用核算我们卡得最细，请研发同事配合。

**适用场景**：为研发项目直接发生的：
- 研发材料：做实验、打样消耗的材料、元器件、原料。
- 测试费：委托外部检测/试验机构的测试、检验费。
- 样机/试制：制作样机、模具、试制品的费用。
- （还有研发人员工资、研发设备折旧、研发差旅等，分别在各自科目按研发口径归集。）

**单据/发票要求**：
- 增值税专用发票（研发材料、测试服务多可抵进项；材料 13%，测试服务 6%）。
- **必须关联具体研发项目号**！在鼎财报销时选研发项目（项目立项后系统里有项目号），不关联项目号的研发支出无法归集加计扣除，等于白花。
- 委托测试要有委托合同/协议；样机试制要有领料单、试制记录佐证。

**税务处理与抵扣**：
- 进项税：研发材料、外部测试取得专票可抵。
- 企业所得税：符合条件的研发费用可**加计扣除**（按现行政策研发费用在据实扣除基础上再加计扣除，制造业研发加计比例高）。**前提是费用归集准确、项目台账齐全、辅助账规范。** 所以研发同事报销时**一定要选对项目号、写清用途**，这是我们税务组年底做研发费用辅助账、应对税务核查的命根子。
- 注意：**委托境外研发、与研发无直接关系的费用**有特殊限制，大额委托研发请提前找税务组确认。

**限额标准 / 审批**：
- 研发材料、测试：走研发专项额度，< 10,000 元 研发项目负责人审批；≥ 10,000 元 研发中心负责人 + 财务复核；大额（≥ 50,000 元）+ 分管领导，且事前预申请。
- 样机/试制大额走采购+合同流程。

**常见错误与不可报情形**：
- 研发支出不选项目号、随便记到"管理费用"——加计扣除归集不到，最大的浪费。
- 把生产领用的材料当研发材料报（研发材料是研发项目领用，不是量产领用）——口径混淆会被税务调减。
- 把研发人员的招待、福利硬塞进研发费用想加计——招待福利不能加计，别塞。
- 项目已结题还继续往里报研发费——退，结题后的支出不归该项目。

**举例**：研发员工王工为"XX新型传感器"项目采购测试元器件一批，含税 12,000 元，专票，鼎财里关联项目号 RD-2026-018，事由写"用于传感器原型第三轮老化测试"，记"研发费用—直接投入—材料"，进项抵扣，年底归集进该项目辅助账参与加计扣除。审批走研发中心负责人+财务复核。

### 2.7 会议费

**适用场景**：召开内部/对外会议租用场地、会议室、会议期间的餐饮（会议餐）、住宿、资料、设备租赁、会务服务等。注意会议费和招待费要分清：会议是有正式议程、有签到、有纪要的；纯粹请人吃饭那是招待。

**单据/发票要求**：
- 会议费专票可抵（会务服务 6%）。
- **必须有会议佐证材料**：会议通知/议程、参会签到表、会议纪要、会议照片中至少几项。**没有会议材料的"会议费"，税务上极易被认定为变相招待或福利，进项要转出，企业所得税也可能被调整。** 这是会议费最大的风险点。
- 委托会务公司办的，要会务合同 + 明细。

**税务处理与抵扣**：
- 会议场地、会务服务取得专票可抵进项。
- 但**会议中的餐饮部分**，如果发票单独开了餐饮，那部分按招待性质处理（不抵、受招待费限额约束）；如果是会务公司打包开"会议服务费"含餐，整体按会议费处理可抵——所以鼓励大型会议让会务公司打包开票，而不是餐饮单独开。
- 企业所得税：合理的会议费据实扣除，但必须有上面说的会议佐证，否则有调整风险。

**限额标准 / 审批**：
- 小型内部会议（场地+茶歇）：< 3,000 元 部门负责人审批。
- 中型会议（≥ 3,000 元）：事业部分管领导审批，需附议程和预算。
- 大型会议/年度会议（≥ 30,000 元）：分管领导 + 财务总监，事前预申请 + 比价选会务商。
- 会议餐人均参照招待标准从严。

**常见错误与不可报情形**：
- 把部门聚餐、团建包装成"会议费"——查实重罚，这是审计高发点。
- 会议费无任何会议佐证材料。
- 会议费里夹带旅游、景点门票、与会议无关的高消费——剔除，景点游览那是变相福利。

**举例**：某事业部办年度经销商大会，委托会务公司含场地、会务、会议餐打包开"会议服务费"专票 86,000 元，附会议通知、议程、签到表、纪要。报"管理费用—会议费"，进项 86000÷1.06×6%≈4,868 元抵扣，走分管领导+财务总监审批，事前已预申请比价。

---

### 2.8 培训费

**适用场景**：员工参加外部培训、考证、研讨班，或公司请外部讲师来内训。包含培训费、必要的培训教材、外训差旅（差旅部分走差旅费）。

**单据/发票要求**：
- 培训费专票可抵（教育/培训服务 6%，部分非营利教育机构可能开免税普票）。
- 要有培训通知/报名记录、培训内容说明；内训请讲师的要有讲师协议、课酬（讲师课酬属劳务报酬，见 2.13）。
- 考证报销要凭准考证/成绩/证书佐证。

**税务处理与抵扣**：
- 外部培训服务取得专票可抵进项。
- **职工教育经费**：用于员工教育培训的支出，企业所得税前按工资薪金总额 **8%** 限额扣除，超出部分结转以后年度。所以集团培训费有总盘子控制，不是无限报。
- 注意：和员工切身利益相关、带有福利性质的"培训"（比如纯团建式的拓展）不能当培训费抵，要按福利处理。

**限额标准 / 审批**：
- 个人外部培训/考证：单次 < 2,000 元 部门负责人审批；≥ 2,000 元 需分管领导 + HR 归口确认（培训与岗位相关性）。
- 公司内训请外部讲师：按金额走招采/合同流程，≥ 30,000 元 财务总监审批。
- 报销前一般需 HR 培训计划内或经 HR 确认（避免与岗位无关的乱报）。

**常见错误与不可报情形**：
- 报与岗位完全无关的兴趣班、学历提升（除非公司有专门的学历资助政策走另外流程）。
- 把拓展旅游当培训费报。
- 个人考证未通过 HR 确认相关性就报。

**举例**：内勤员工小郑参加"中级会计职称考前辅导班"，培训机构开专票 1,800 元，附报名记录，经部门负责人审批，HR 确认与岗位相关，报"管理费用—职工教育经费—培训费"，进项可抵，年底纳入职工教育经费 8% 限额核算。

---

### 2.9 通讯补贴

**适用场景**：因工作需要使用个人手机/宽带产生的通讯费用，集团按职级发放通讯补贴。

**单据/发票要求**：
- 集团现行**通讯补贴按月随工资发放、定额包干，不需要发票**（这是政策性补贴，简化处理）。**因此通讯补贴并入工资，需要按规定计征个人所得税。**
- 如果是公司统一办理的对公手机号/集团套餐（运营商对公开票给公司），那走"管理费用—通讯费"，专票可抵进项，与个人补贴是两回事。

**税务处理与抵扣**：
- 个人通讯补贴（随工资发的定额）：**并入工资薪金，计征个税**，企业所得税作为工资薪金支出扣除。无进项可抵（没发票）。
- 对公通讯费（运营商开给公司的）：专票可抵进项（基础电信 9%，增值电信 6%）。

**限额标准**（集团现行，每月定额）：
- 普通员工：80 元/月。
- 主管/经理：150 元/月。
- 总监及以上：300 元/月。
- 销售/外勤岗位经批可上浮。
- 外包员工一般无通讯补贴（除非合同/项目另有约定）。

**常见错误与不可报情形**：
- 拿个人手机话费发票再单独来报销——错，已经发定额补贴了不能再报，重复。
- 想用个人话费发票抵公司进项——个人抬头的发票不能抵公司进项，也不该报。

**举例**：经理级老孙每月通讯补贴 150 元，随 4 月工资发放，已并入工资计个税，无需任何发票，无需在鼎财单独报销。

---

### 2.10 福利费（职工福利费）

**适用场景**：员工集体福利、节日慰问、生日礼品、防暑降温、困难补助、体检、食堂补贴、过节费、工会以外的集体福利等。

**单据/发票要求**：
- 福利采购可以拿发票（普票即可，**因为职工福利的进项税不得抵扣**，开专票还要转出，所以开普票）。
- 节日礼品、慰问品要有发放记录/签收名单佐证（谁领了，多少）。

**税务处理与抵扣**：
- **职工福利费进项税不得抵扣**（用于集体福利/个人消费的购进，进项转出）。所以别要专票。
- 企业所得税：职工福利费按工资薪金总额 **14%** 限额扣除，超出调增。
- **个税**：很多福利要并个税！普惠性的、不可分割的集体福利（如集体食堂、防暑降温饮品）一般不并个税；但发给个人的、可量化到人的实物/现金福利（过节费、购物卡、可分割的礼品），原则上要并入工资薪金计个税。**这点最容易被忽略**，发购物卡、过节红包不代扣个税是稽查高发问题，财务会按规定处理，请部门配合提供发放名单。

**限额标准 / 审批**：
- 节日福利：集团统一标准，逢法定节日人均 ≤ 200–500 元（视节日），由工会/行政统一采购发放，部门一般不单独报福利。
- 部门小范围福利（团队生日会等）需部门负责人审批，单次有上限。
- 体检：按集团统一体检方案，不走个人报销。

**常见错误与不可报情形**：
- 把福利伪装成办公用品/会议费想抵进项——查实转出并追责。
- 发购物卡不报送名单导致个税漏缴。
- 用福利费名义发与全员无关的小团体高额消费。

**举例**：行政统一采购端午节慰问品（粽子礼盒）全员发放，开普票 4.5 万元，附全员签收名单，记"应付职工薪酬—职工福利费"，进项不抵，年底纳入福利费 14% 限额；因属可分割实物福利，按规定并入个人当月工资计个税。

---

### 2.11 团建费

**适用场景**：部门/团队团队建设活动，如团队聚餐、户外拓展、短途团建、节日活动等。**注意：团建对象是本部门员工自己，不是客户**（请客户是招待）。

**单据/发票要求**：
- 团建多含餐饮、活动场地、拓展服务等。餐饮普票即可（不抵）；拓展/场地服务能开专票，但因团建整体偏福利性质，进项抵扣要谨慎，财务会按实质判断。
- 必须有**团建申请 + 参加人员名单 + 活动说明**（什么活动、几人参加、人均多少）。

**税务处理与抵扣**：
- 团建费在税务上多按**职工福利费**性质处理（受 14% 限额约束），餐饮、个人消费部分进项不抵。
- 如果团建实质是公司组织的、提升团队协作的合理支出，企业所得税可在福利费限额内扣除。
- 团建中可分割到个人的福利（如发给个人的纪念品）注意个税。

**限额标准 / 审批**（集团现行）：
- 团建人均：≤ 300 元/人/次（含餐饮活动），重要项目庆功经分管领导批可 ≤ 500 元/人。
- 频次：每个团队每季度原则上 1 次，每年不超过 4 次。超频次的不予报销。
- 单次团建总额 ≥ 5,000 元 需部门负责人 + 分管领导审批。
- 团建不得报酒水中的高档酒、KTV 等娱乐性消费（按集团规定从严）。

**常见错误与不可报情形**：
- 把团建包装成"会议费"想抵进项——审计高发，别这么干。
- 团建带家属、带外部人员的费用混报。
- 超频次、超人均还硬报。
- 把个人或小圈子吃喝当团建报。

**举例**：研发某组 12 人季度团建，聚餐+桌游共 3,480 元，人均 290 元（未超 300），当季首次（未超频次），附名单和活动说明，报"职工福利费—团建"，部门负责人审批，进项不抵，纳入福利费限额。

---

### 2.12 外包服务费 / 劳务费

这块容易混，先把两个概念分清：
- **外包服务费**：付给一家有资质的**公司**（供应商）提供的服务，比如 IT 系统外包、保洁外包、设计公司、咨询公司、人力外包公司。对方是企业，开**增值税发票**。
- **劳务费（劳务报酬）**：付给**个人**（自然人）的报酬，比如外部讲师课酬、临时聘请的个人专家、个人提供的零星劳务。涉及**代扣代缴个人所得税（劳务报酬）**，和发工资不一样。

**适用场景**：
- 外包服务费：公司委托外部企业完成的服务（开发、设计、咨询、保洁、安保、人力外包、运维等）。
- 劳务费：个人为公司提供一次性/临时性劳务（讲课、评审、临时翻译、个人技术服务）。

**单据/发票要求**：
- 外包服务费：对方公司开**增值税专用发票**（多为现代服务 6%，可抵进项），要有**服务合同 + 验收/成果确认 + 供应商信息**。大额外包对公付款（不是个人报销）。
- 劳务费（付给个人）：个人需到税务机关**代开发票**，或公司按规定**代扣代缴个人所得税**后支付。没有发票又不代扣个税直接发现金，是违规的。要有劳务协议、成果证明、个人身份信息和银行卡。

**税务处理与抵扣**：
- 外包服务费：取得专票可抵进项（6% 居多）。企业所得税据实扣除。
- 劳务费：付给个人的劳务报酬，公司有**代扣代缴个税**义务（劳务报酬按次/按月预扣，年度并入综合所得汇算）。取得个人代开的发票方可税前扣除（小额零星个人劳务有免开发票的特殊情形，但需符合条件，金额小，多数还是要发票）。

**限额标准 / 审批**：
- 外包服务费按合同走采购+合同审批，金额大，多由对公付款而非个人报销。≥ 50,000 元 财务总监审批，≥ 一定金额 需走招采比价。
- 劳务费：单次个人劳务 < 5,000 元 部门负责人 + 财务审批；≥ 5,000 元 加分管领导，且必须代扣个税。

**常见错误与不可报情形**：
- 付给个人的劳务费**不代扣个税**就发——红线，税务稽查重点。
- 拿不到外包公司发票就用个人发票顶替——不行。
- 把本应签劳动合同的长期用工伪装成"劳务费"规避社保个税——合规风险，HR+财务会管。
- 外包成果无验收、无合同就付款。

**举例**：集团请外部专家张教授来做一次技术评审，约定课酬 8,000 元，张教授到税务局代开劳务发票，公司按劳务报酬代扣个税后支付。记"管理费用—劳务费"，附劳务协议、评审纪要。因 ≥ 5,000 元，走到分管领导审批。

---

### 2.13 快递与印刷费

**适用场景**：
- 快递费：寄送合同、样品、文件、物料的快递邮寄。
- 印刷费：名片、宣传册、标书、画册、资料印刷、装订。

**单据/发票要求**：
- 快递专票可抵（物流/快递服务 6% 或运输 9%）。
- 印刷专票可抵（印刷服务 13% 或 6%，看具体）。
- 印刷大批量宣传品要有用途说明（避免变成给客户的礼品而被归为招待）。

**税务处理与抵扣**：
- 快递、印刷取得专票均可抵进项。
- 注意：如果印刷的是给客户的礼品性质宣传品、或快递寄送的是给客户的礼品，那对应的礼品本身可能涉及招待，但快递/印刷服务本身的进项一般可抵。

**限额标准 / 审批**：
- 快递费据实，部门内审批；月度异常高的要说明。
- 印刷费单次 < 2,000 元部门审批；≥ 2,000 元部门负责人 + 行政归口；大批量宣传品集中印刷走采购。

**常见错误与不可报情形**：
- 私人寄件混入公司快递报销。
- 印刷无用途说明、品名笼统。

**举例**：市场部印刷产品画册 500 份，印刷厂开专票 3,600 元，附用途说明"用于4月华南展会发放"，报"管理费用—印刷费"，进项 3600÷1.13×13%≈414 元抵扣，走部门负责人+行政归口审批。

---

### 2.14 软件订阅费 / 信息服务费

**适用场景**：订阅 SaaS 软件、云服务、办公协同工具、设计软件、数据库/资讯服务、域名、服务器租用等。研发用的开发工具、云算力也可能走这里或研发费用。

**单据/发票要求**：
- 增值税专用发票（软件/信息技术服务 6%，可抵进项）。境外软件订阅（如某些海外 SaaS）取得不了境内发票的，涉及**代扣代缴增值税及附加（境外服务）**，要走特殊流程，请提前找税务组，别个人刷卡了事。
- 要有订阅说明（什么软件、订阅周期、账号数）。

**税务处理与抵扣**：
- 境内软件/信息服务取得专票可抵进项。
- 境外服务：可能需要代扣代缴增值税（境外单位在境内未设机构的，购买方代扣），这块由税务组处理，个人不要直接用私人卡支付境外订阅再来报销，很难合规。
- 资本化判断：大额、长期的软件（如外购的、构成无形资产的软件系统）可能要作为无形资产摊销，由财务判断；普通的月度/年度 SaaS 订阅一般费用化。

**限额标准 / 审批**：
- 个人/团队订阅工具：< 2,000 元/年 部门审批；≥ 2,000 元 部门负责人 + IT 归口（避免重复采购、安全合规）；集团级软件采购走 IT + 采购流程。
- 研发工具订阅挂研发项目的归研发费用。

**常见错误与不可报情形**：
- 个人用私人信用卡（尤其外币卡）订阅境外软件再报——抵扣和代扣个税/增值税都难处理，强烈不建议。
- 同一软件多人重复订阅未走 IT 归口。
- 把个人娱乐订阅（视频会员等）混报。

**举例**：研发团队订阅某代码托管/CI 云服务年费，境内代理商开专票 6,800 元/年，关联研发项目，经 IT 归口确认，报"研发费用—信息服务"或"管理费用—软件服务费"（按是否挂研发项目），进项可抵。

### 2.15 其他常见小科目速查

为节省篇幅，下面这些用得少但也会遇到的，简表速查：

| 科目 | 适用 | 发票要求 | 能否抵进项 | 提醒 |
|---|---|---|---|---|
| 邮电费 | 公司对公电话/邮政 | 专票 | 可抵 | 与个人通讯补贴区分 |
| 水电物业费 | 办公场地 | 专票 | 可抵（注意分摊） | 多为对公付款 |
| 车辆使用费/油费 | 公车加油、过路费 | 专票/通行费数电票 | 可抵 | 私车公用另有规定 |
| 银行手续费 | 对公账户手续费 | 银行回单/发票 | 一般不涉个人报销 | 财务直接处理 |
| 工会经费 | 工会活动 | 按工会规定 | 不抵 | 走工会账，不混福利 |
| 保险费 | 公司给资产/责任投保 | 专票 | 部分可抵 | 给员工的商业保险注意个税 |
| 招聘费 | 招聘渠道、猎头 | 专票 | 可抵（服务6%） | 给候选人的差旅不走员工差旅 |

## 第三章 不同报销场景，分别走哪些科目、怎么注意

前面是按"科目"切的，现实里大家是按"场景"报销的。这一章按场景串一遍，帮你把科目对应起来。

### 3.1 场景：本地日常办公

**典型支出与对应科目**：
- 本地因公打车/地铁 → 2.3 市内交通费
- 买文具耗材 → 2.4 办公用品费
- 寄合同/文件 → 2.13 快递费
- 印名片/资料 → 2.13 印刷费
- 订办公软件 → 2.14 软件订阅费
- 加班晚归打车 → 2.3 市内交通（加班交通，需加班记录）

**注意**：本地日常最容易混的是"本地交通"被误填成"差旅费"。差旅是出差，本地不是出差。还有就是办公用品无清单退单，记得让供应商出销货清单。

### 3.2 场景：出差（最复杂，重点）

一次出差通常一张主报销单，下挂多条明细，分别走差旅费的子项：
- 大交通（机票/高铁）→ 2.1.1
- 住宿 → 2.1.2（尽量要专票抵扣）
- 出差期间市内交通 → 2.1.3
- 出差补贴（伙食杂费，不要票）→ 2.1.4

**全流程注意**：
1. 出差前先提**出差申请**（≥5,000 元走预申请，外包无论金额都要事前批）。
2. 行程、城市、日期要和申请一致。
3. 补贴和餐费二选一，别重复。
4. 出差顺带请了客户吃饭——那顿是**招待费**（2.2），单独拆出来写明招待对象，别混进自己的差旅。
5. 研发员工出差做项目相关的事，差旅要挂研发项目号，归集研发费用差旅。
6. 出差垫付较多的，回来 90 天内报完。

> 邮件片段（财务退单常见话术，给大家感受一下）：
> "您好，您 4 月深圳出差单：①住宿发票为普票，建议下次要专票以便抵扣（本次已受理）；②4月16日有一笔餐饮 380 元，与当日出差补贴重复，请二选一，已退回该明细，请删除后重新提交；③有一笔 620 元餐费疑似招待，请补充招待对象信息或转至招待费科目。——财务核算组 小韩"

### 3.3 场景：招待客户

**对应科目**：2.2 业务招待费（餐饮、礼品）。涉及的住宿/交通如果是为招待对方产生的，也按招待性质（不抵），别混进自己差旅。

**注意**：
- 事由必写招待对象单位、人数、我方陪同人数。
- 餐饮开普票即可（不抵）。
- 金额按阈值升级审批，≥5,000 要事前预申请。
- 别化整为零拆票。
- 高档酒水、贵重礼品从严。

### 3.4 场景：采购（办公/设备/物料）

**对应科目**：
- 小额办公消耗 → 2.4 办公用品费
- 高价值长期使用的设备 → 2.5 设备采购（注意固定资产 5,000 元标准、要专票、关联合同验收）
- 研发专用材料/设备 → 2.6 研发费用 / 研发设备额度

**注意**：
- 大额采购**走采购流程对公付款**，不是个人垫付报销。
- 专票品名要明细、附清单。
- 别拆票规避固定资产标准或审批阈值。

### 3.5 场景：研发项目

**对应科目**（全部要挂研发项目号）：
- 研发材料、元器件 → 2.6 研发材料
- 委外测试检验 → 2.6 测试费
- 样机试制 → 2.6 样机
- 研发专用设备 → 2.5/2.6 设备
- 研发人员出差 → 2.1 差旅（挂研发项目）
- 研发用软件/云算力 → 2.14（挂研发项目）

**注意**：
- **选对项目号是命根子**，关系加计扣除。
- 不要把量产领料、与研发无关的费用塞进研发项目。
- 项目结题后不再往里报。
- 大额委托研发、境外研发提前找税务组。

### 3.6 场景：培训与会议

**对应科目**：
- 员工外训/考证/内训 → 2.8 培训费（受职工教育经费 8% 限额；要 HR 确认相关性）
- 办会议（场地、会务、会议餐打包）→ 2.7 会议费（必须有议程、签到、纪要佐证）
- 会议涉及的外训差旅 → 2.1 差旅

**注意**：
- 培训别报与岗位无关的兴趣班。
- 会议费最怕"无会议材料"被认定为变相招待/福利，材料一定齐。
- 会议餐尽量让会务公司打包开"会议服务费"，别餐饮单开。

### 3.7 场景：外包结算 / 付个人劳务

**对应科目**：
- 付外包公司服务 → 2.12 外包服务费（对方公司开专票，合同+验收，多对公付款）
- 付个人劳务/讲师课酬 → 2.12 劳务费（代扣个税，个人代开发票）

**注意**：
- 外包公司服务尽量要专票抵进项。
- 付个人**必须代扣个税**，不能现金白条。
- 长期用工别用劳务费规避社保。

### 3.8 场景：外包/驻场员工自己报销（特别提醒）

外包/驻场同事看这里：你们能报的科目**很有限**，而且**必须先经甲方项目经理或对接人审批**，财务才受理。一般你们可报的：
- 项目要求的、经甲方批准的现场公务交通（很多项目是包干的，确认清楚）。
- 经甲方批准的特定项目支出。

**不能报的（除非合同/项目特别约定）**：通讯补贴、福利费、团建费、本地日常通勤、个人办公用品等。报销前先问清楚你的项目对接人和我们财务对接窗口，别先垫钱后发现报不了。

## 第五章 审批阈值与时限统一汇总

把散落在各科目的审批阈值和时限集中放这里，方便查。**业务线和财务线两条线，金额越大、科目越敏感，往上走得越高。**

### 5.1 业务审批线（按单笔金额升级）

| 金额区间 | 业务审批到哪一级 |
|---|---|
| < 2,000 元 | 直属上级（部门内）|
| 2,000 – 20,000 元 | 直属上级 → 部门负责人 |
| 20,000 – 50,000 元 | 加 事业部分管领导 |
| ≥ 50,000 元 | 分管领导（敏感科目再加财务总监）|

### 5.2 财务审批线（按金额/科目升级）

| 情形 | 财务审批到哪一级 |
|---|---|
| 一般小额、常规科目 | 费用会计初审即可 |
| 中额或需复核科目 | 费用会计初审 → 财务复核 |
| 大额 / 敏感科目（招待、设备、外包、研发大额、会议大额）| 加 财务经理 / 财务总监终审 |

### 5.3 事前预申请阈值（发生前就要批）

| 员工类型 | 需事前预申请的情形 |
|---|---|
| 内勤员工 | 单次费用 ≥ 5,000 元 |
| 研发员工 | 研发专项/设备 ≥ 10,000 元；差旅 ≥ 5,000 元 |
| 外包员工 | 任何费用发生前都要预申请（无论金额）|

### 5.4 时限

- 发票提交：开票之日起 **90 天内**报销。
- 跨年发票：最晚次年 **1 月 15 日**前报完（年度结账前）。
- 紧急出差事后补单：**3 个工作日内**补出差申请，双签说明。
- 财务退单后补正：收到退单 **5 个工作日内**补正重提，超期需说明。

## 第六章 高频踩坑总结（财务一年退单退到想哭的那些）

最后把我们退单最多、最想跟大家念叨的，列个清单，记住能省大家很多返工：

1. **办公用品/材料"无销货清单"** —— 退单第一名。让供应商系统出清单。
2. **该要专票的要了普票**（住宿、办公、设备、会议、外包、研发）—— 抵不了进项，集团多交税。
3. **不该要专票的偏要专票**（招待、福利、团建）—— 要了还得转出，徒增麻烦还暴露问题。
4. **出差补贴和餐费重复报** —— 二选一。
5. **本地交通填成差旅费** —— 本地不是出差。
6. **招待事由不写对象** —— 必写对方单位、人数、我方陪同。
7. **研发费用不挂项目号** —— 加计扣除归集不到，白白浪费税收优惠。
8. **会议费没有会议材料** —— 被认定变相招待/福利，进项转出。
9. **设备拆票规避固定资产标准** —— 红线。
10. **付个人劳务不代扣个税** —— 税务稽查重点，红线。
11. **发购物卡/过节实物福利不报送名单、不并个税** —— 个税漏缴风险。
12. **境外软件用私人卡刷了再报** —— 抵扣和代扣都难处理，提前找税务组。
13. **化整为零拆票规避审批阈值** —— 审批权限按累计算，查实违规。
14. **发票超期（>90天、跨年未及时报）** —— 错过抵扣期、要走以前年度调整。
15. **抬头/税号写错、写成个人** —— 直接退，从源头让对方按集团开票信息开。

### 第七章 那些"长得像、其实不是一回事"的易混科目（边界判据）

去年正文里讲了招待 vs 宣传、招待 vs 福利、福利 vs 团建、会议 vs 培训、咨询 vs 外包 vs 劳务这几对。今年发现还有一批更隐蔽的，走错了后果还更严重，集中补在这里。每一对我都给"硬判据 + 走错的后果"，你拿不准的时候照着这张表对一遍。

#### 7.1 咨询费 vs 中介服务费 vs 市场推广费

这仨最近被混得最厉害，因为发票上经常写得很笼统，对方一句"我给你开咨询费吧"就把你坑了。

| 科目 | 判据（钱花在哪、买的是什么） | 金蝶里挂哪 | 能否抵进项 | 后果提醒 |
|---|---|---|---|---|
| 咨询费 | 买的是**智力成果/专业意见**：管理咨询、法律咨询、税务咨询、技术方案咨询，有报告、有方案、有结论 | 管理费用—咨询费 | 专票可抵 6% | 必须有书面成果（报告/方案/意见书），光一张发票没成果的，按"无实质"从严 |
| 中介服务费 | 对方是**居间撮合、牵线搭桥**：房产中介、招投标代理、专利代理、人才中介（猎头）、报关代理 | 管理费用—中介服务费（猎头进招聘费） | 专票可抵 6% | 要有居间/代理协议，写清楚撮合的是哪笔事；别和咨询费混，中介是"促成交易"，咨询是"给意见" |
| 市场推广费 | 花在**推广产品、获取客户、品牌曝光**：线上投流、展会、广告、推广活动、KOL 合作 | 销售费用—市场推广费 | 专票可抵（多 6%，广告也可能 6%） | 这是销售费用，不是管理费用，**记错大类影响部门费用率分析**；面向不特定公众的批量派发归这里（不是招待） |

判断口诀：**"买脑子"是咨询，"牵线"是中介，"打广告拉客户"是推广。** 举个真实的，去年某事业部请了家公司帮忙搞了场客户答谢酒会，对方开"咨询费"，经办人就照咨询费报了。我们一看附件是个活动执行清单加几张现场照片，哪有什么咨询报告？这分明是市场推广（甚至带招待性质）。退回去要求按市场推广费重走，里面给到场客户的餐饮那部分还得单独拆出来按招待处理。**对方开什么票不代表你就该按什么科目记，看实质。**

#### 7.2 赞助费——基本不能税前扣除，能不报就别报

这个单拎出来讲，因为踩雷代价最大。

什么叫赞助费？给个会议、给个赛事、给个活动、给个机构掏钱冠名或支持，但**对方不给你提供等价的、与你经营相关的服务**，就是赞助。

关键点：**赞助性支出（非广告性质的）在企业所得税前一律不得扣除。** 也就是说你花的这笔钱，做账归"营业外支出—赞助支出"，年底汇算清缴的时候要**全额纳税调增**，相当于公司白掏钱还多交一道税。进项当然也别想抵。

那"赞助"和"市场推广/广告"怎么分？看一条：**有没有给到公司可衡量的广告/推广回报，并且有合同约定。** 比如你赞助一个行业论坛，合同写明"在主会场背景板、会刊、官网放置鼎垣智造 logo，提供 30 分钟主题演讲时段"——这是**广告性赞助**，能按广告费税前扣除、专票能抵。反过来，你就是单纯给某协会捐了 5 万"支持一下"，没合同、没回报、没 logo 露出——纯赞助，不得扣除。

> 真实踩坑：某事业部为了维护关系，赞助了客户搞的一个内部年会 3 万元，没签任何带露出条款的合同，对方开了张"赞助费"普票。这笔钱我们只能进营业外支出全额调增。当时业务同事还不理解，说"这不也是为了业务嘛"。是为了业务没错，但**税法认的是"有没有合同约定的广告回报"，不认你的心意。** 以后凡是想"赞助"的，先找市场部把它做成带露出条款的广告/推广合同，否则就是纯掏钱。

#### 7.3 会员费——分清是"行业协会会费"还是"买服务"

会员费也分两种，差别很大：

- **行业协会、商会的会员会费**：交给依法成立的社会团体的会费，凭对方开的**财政性票据（行政事业单位资金往来结算票据或符合规定的票据）**入账，**不是增值税发票，不能抵进项**，企业所得税可据实扣除（但有的协会会费税法上有扣除口径，大额的找税务组确认）。挂"管理费用—会费"。
- **平台/数据库/俱乐部的会员费**（本质是买服务）：比如付费数据库会员、行业资讯平台年费、企业级软件的会员订阅——这其实就是信息服务费/软件订阅费，对方开**增值税专票，可抵 6%**，挂"管理费用—软件服务费/信息服务费"（参见正文 2.14）。

判断口诀：**交给协会"维持会员身份"的，是会费、拿财政票据、不抵；付平台"换取服务"的，是服务费、拿专票、可抵。** 别一看见"会员"俩字就都往一个科目塞。

#### 7.4 罚款、滞纳金、违约金——三个长得像，处理完全不同

这三个年年有人搞混，今年专门拉个表说清楚：

| 类型 | 例子 | 能否税前扣除 | 能否抵进项 | 备注 |
|---|---|---|---|---|
| 行政性罚款、罚金、滞纳金 | 税收滞纳金、交通违章罚款、环保罚款、工商罚款、社保滞纳金 | **一律不得税前扣除** | 不涉及（没有发票，是行政收据/缴款凭证） | 营业外支出，年底全额纳税调增 |
| 银行罚息 / 逾期利息 | 贷款逾期罚息 | 可税前扣除 | 不涉及 | 这是经营性利息，不是行政罚款 |
| 经营性违约金 / 赔偿金 | 合同违约赔对方、退货补偿 | 一般可扣除（要有合同、协议、对方收据） | 看是否随价款、是否开票 | 跟交易直接相关的可扣；纯赔偿对方损失、对方开不了票的凭协议+收款凭证 |

最常见的雷：**税收滞纳金、各种行政罚款不能税前扣除，更不要拿这种凭证来想抵进项。** 司机贴个交通违章罚款单来报，能给你报销（公司承担）是一回事，但记账只能进营业外支出且年底调增，绝不能混进"差旅费—市内交通"去稀释，更不能抵进项。**罚款滞纳金这一类，记住六个字：不抵、不扣、单列。**

至于经营性违约金，比如我们晚交货赔了客户 2 万，要有合同违约条款、双方确认的赔偿协议、对方的收款凭证，这种才能税前扣，且要看对方能不能开票（不随价款的损害赔偿对方往往开不了增值税发票，凭协议入账）。这类大额的报上来先找我们税务组过一遍。

#### 7.5 几对边界，一句话速记（贴在工位上那种）

- **咨询费 vs 中介费**：给意见出报告是咨询，撮合交易是中介。
- **赞助费 vs 广告费**：有合同约定露出/回报是广告（能扣能抵），单纯掏钱是赞助（不扣不抵，全额调增）。
- **会费 vs 服务费**：交协会维持身份是会费（财政票据不抵），付平台换服务是服务费（专票可抵）。
- **罚款滞纳金 vs 违约金**：行政罚款滞纳金不扣不抵单列；经营性违约金凭合同协议可扣。
- **赔偿金对内 vs 对外**：赔给客户/供应商的看合同；赔给自己员工的（如解约补偿）走应付职工薪酬，是另一套口径。

### 第八章 这些科目在金蝶里大致怎么记（给业务看，看懂为什么卡得严）

经常有同事来问："我这单不是早就审批通过了吗，怎么钱还没到账？"或者"你们财务说我'已入账'，那钱呢？"——这是把"做账确认费用"和"实际付款"当成一回事了。其实是两笔账。这一节用几个示意分录给业务同事看明白，看完你就知道我们为什么有些环节卡得严。

> 说明：下面的分录是**示意**，金额、税率以实际为准。看不懂借贷没关系，重点看"哪一步费用确认了""哪一步钱出去了"。

#### 8.1 一笔普通差旅报销，从入账到付款是两笔账

王工报销高铁票 1,090 元（前面正文 2.1.1 那个例子），财务这边实际上分两步走。

**第一步，审批通过、费用会计做账（我们叫"已入账"）：**

```
借：管理费用—差旅费          1,000
借：应交税费—应交增值税（进项税额）  90
    贷：其他应付款—应付职工报销款（王工）  1,090
```

到这一步，**费用确认了（进了管理费用）、进项也确认了（90 元进项挂上了）**，但钱还没出去——挂在"其他应付款"里欠着王工 1,090 元。所以系统显示"已入账"的时候，账上确实记了，但你卡里还没钱。

**第二步，资金部按付款批次实际打款（我们叫"已付款"）：**

```
借：其他应付款—应付职工报销款（王工）  1,090
    贷：银行存款                    1,090
```

这一步钱才真出去，"其他应付款"清掉，王工到账。

**所以"已入账 ≠ 已付款"**，中间隔着付款批次的截单和银行处理。这就是为什么有同事周二上午审批过了，钱不是当天到——它得等下一个付款批次。这块的截单时点和批次顺延规则，01 那本《员工报销操作手册》里讲了，这里就不重复。我们想让业务明白的是：**费用我们这边账上早记了（影响你们部门的费用率和预算占用），跟钱什么时候到是两件事。**

#### 8.2 进项转出——为什么"招待、福利别要专票"是硬要求

正文里反复说招待、福利不能要专票，很多人觉得"我要了专票放着不抵不就行了？"——不行，麻烦就麻烦在这儿。

假设某同事请客户吃饭，财务图省事或者对方主动开了张餐饮**专票**，1,060 元，而且这张专票在勾选平台上**勾选认证了**（一旦认证，这 90 多块的进项就进了当期增值税申报表）。后来核算组复核发现这是招待，按规定招待进项不得抵扣，必须做**进项转出**：

```
借：管理费用—业务招待费        60
    贷：应交税费—应交增值税（进项税额转出）  60
```

（金额为示意）这一转，相当于把本来抵掉的进项又加回来交税，**当期增值税应纳税额被拉高**。更要命的是，专票一旦认证就进了金税系统的比对链条，你这边又做转出，**金税系统会留下"认证后转出"的痕迹，是税务关注的预警点之一**。

所以"招待、福利、团建别要专票"不是嫌麻烦，是：**要了专票一认证就上了申报表，再发现是招待福利必须转出，既拉高当期税负，又给自己挖个预警坑。** 这几类直接开普票，从源头就没进项可抵，干干净净。

#### 8.3 暂估、待摊、计提在金蝶里长什么样（业务知道有这回事就行）

这块属于权责发生制的处理，制度增补（02 那本）讲透了概念，这里只给业务同事看一眼"账上是怎么挂的"，方便你理解为什么财务会让你"先提个计提单""填服务起止期间"。

- **暂估**（东西收到用了、发票没来）：先按合同金额估着入成本/费用，挂"应付账款—暂估"，发票来了再红冲暂估、按实际入账。所以你报销单上关联了原暂估单号，系统就自动冲销了，别重复入。
- **待摊**（一次付一大笔、服务好几个月）：比如年付的软件订阅 36,000 元，先挂"预付账款/长期待摊费用"，再按月一笔笔摊进费用。**这就是为什么你报年度订阅必须填"服务起止期间"**——不填，系统不知道摊几个月，只能当月一次性进费用，被退回。
- **计提**（活干了、发票钱都没到，但归这个月/这一年）：年底常见，先借费用、贷"预提费用/其他应付款"，发票来了冲掉。

业务同事不用会做这些分录，只要记住一点：**财务让你提计提单、填服务期间、关联暂估号，都是为了让费用记到它该归的那段时间去，不是给你找麻烦。**

### 第九章 进项抵扣的几种特殊情形（容易出错、容易踩雷）

正文第四章那张快速对照表把"能不能抵"列清楚了。但实操里有几种特殊情形，光看表不够，今年专门补一补。

#### 9.1 价税分离——员工一律填价税合计，别自己拆

经常有较真的同事问："我这张专票价 1,000、税 130，我报销金额填 1,000 还是 1,130？"——**统一填价税合计 1,130（票面含税总额）。** 拆分价税、算进项是我们核算组的活，系统会按你填的发票号、税额自动处理。你自己拆反而容易拆错、对不上。

为什么强调？因为价税分离是抵扣的基础动作：可抵的，含税额拆成"价"进费用、"税"进进项；不可抵的（招待福利），价税合计整个进费用，那 90 块税也变成你的成本。所以**同一笔钱，能不能抵，最终公司承担的成本是不一样的**——可抵的实际成本是不含税价，不可抵的实际成本是含税全额。这也是我们反复念叨"该抵的别漏抵"的原因。

#### 9.2 小规模供应商开的专票

现在不少小供应商是小规模纳税人，他们开的专票税率是 1% 或 3%（不是一般纳税人的 6%/13%）。这种票**照样能抵**，按票面注明的税额抵。员工还是那句话：**填价税合计，进项按票面税额抵，拆分我们做。** 别因为税率低就觉得"是不是不能抵"而要求对方换票，能抵的别折腾。

#### 9.3 混开发票——按品名拆成多行，只抵能抵的

一张发票里既有能抵的、又有不能抵的，这种正文 1.2 提过，这里说透处理口径。

最典型：差旅住宿的全电票里，住宿费（6% 可抵）和早餐/餐饮（招待性质、不可抵）开在一张票上，或者会务公司一张票里既有会议服务（可抵）又单列了餐饮。处理方式是**按品名拆成多行分别选科目**：能抵的那行按可抵处理，不能抵的那行按不可抵处理，**系统只对能抵的部分算进项**。

还有携程这类平台的票，住宿费和"代订服务费"是两码事：服务费是平台收的现代服务（6% 可抵），住宿费按住宿口径，**分两行报，别拿服务费冒充住宿费，也别整张当住宿报。**

#### 9.4 通行费、桥闸费——能抵的别漏，桥闸的别错抵

通行费数电票可以抵：高速公路通行费按 **3%** 计抵，一级二级公路通行费按 **5%**——不对，集团口径统一按取得的**通行费电子发票上注明的税额**抵（数电票直接显示可抵税额，照着抵）。重点提醒一条：**桥、闸通行费不可抵进项！** 这是税法的特殊规定，桥闸通行费和普通公路通行费不是一回事。司机或公车报通行费的时候，财务会把桥闸那部分单独剔出来不抵，别奇怪。

#### 9.5 视同销售——这几种"没收钱也要按销售交税"的情形

"视同销售"是业务同事最容易忽略的一块，因为它"没收到钱、却要交税"，反直觉。说人话：**有些东西虽然没卖、是送出去或自用了，但税法把它当成卖了，要按视同销售交增值税（有的还涉及企业所得税）。**

跟报销沾边的常见情形：

- **把自产产品/外购货物当礼品送客户**：比如把我们自己造的产品送给客户做样品/礼品，增值税上要视同销售、按市场价计销项税。**所以这类"送货出去"的事，别自己在报销里随手处理，一定走财务，由我们做视同销售。**
- **把外购货物用于职工福利、个人消费**：比如买了一批东西发给员工当福利，这批货的进项不能抵（要转出），处理上跟视同销售相关。
- **无偿赠送给外部单位/个人的货物**：同理要视同销售。

业务要做的就一件事：**凡是涉及"把货物/产品无偿送出去"或"自家产品自用"的，提前报财务税务组，别自己在报销单里消化。** 视同销售算错，增值税和企业所得税两头都可能出问题。

#### 9.6 不可抵扣情形，再钉一遍清单

把不能抵的集中钉一遍，免得各章翻来翻去：

1. **用于招待、集体福利、个人消费**的购进（餐饮、福利品、员工旅游团建）——不抵，要了专票还得转出。
2. **餐饮服务**——餐饮的进项本身就不让抵（哪怕不是招待，比如加班餐），所以餐饮类基本都开普票。
3. **取得的不合规发票**（假票、品名不符、抬头错、过认证期）——不抵。
4. **桥、闸通行费**——不抵（见 9.4）。
5. **行政罚款、滞纳金的凭证**——根本不是增值税发票，谈不上抵（见 7.4）。
6. **个人抬头的发票**——不抵（见第十二章）。
7. **简易计税、免税项目对应的进项**——不抵，但这个一般到不了员工层面，财务处理。

### 第十章 资本化 vs 费用化（你只管标清楚，判断归财务）

正文 2.5 讲了固定资产 5,000 元门槛，这里把"资本化还是费用化"这个更大的话题补全，重点是研发、软件、装修、拆分采购这几个高频争议。先把铁律放最前面：

> **资本化还是费用化，由财务（核算组 + 研发会计）统一把口，业务和研发同事不要自己判断、自己拆。你要做的是在报销时把信息标清楚——什么东西、单价、用途、研发的话挂哪个阶段——剩下的我们来定。**

#### 10.1 研发的研究阶段 vs 开发阶段

研发支出在会计上分两段，处理完全不同：

- **研究阶段**（探索、调研、试验、可行性论证，结果还不确定）：**全部费用化**，当期进研发费用。
- **开发阶段**（已经过了研究、技术可行、有明确商业用途、成本能可靠计量）：满足条件的可以**资本化**，形成无形资产，以后分期摊销。

研发同事报销时**标注清楚处于哪个阶段**（系统里选），但**别自己下结论说"这个该资本化"**——是不是真满足资本化的三个条件（技术可行、明确商业用途、可靠计量），由研发会计结合项目立项材料判断。标错阶段会影响当期利润和加计扣除，这块我们盯得很紧。

#### 10.2 "能一次性税前扣除"≠"不用入固定资产"

这是去年争议最大的一个误区，必须讲清楚。

税法有个优惠：单价 ≤ 500 万元的设备、器具，企业所得税上可以选择**一次性税前扣除**（不用按年慢慢折旧扣）。于是有同事就说："那我这台 8 万的设备能一次性扣，是不是就不用入固定资产、直接当费用报掉？"

**错！这是把"会计处理"和"税务处理"搞混了。**

- **会计上**：单价 ≥ 5,000 元、用超 1 年的，照样**入固定资产、建卡、按年限折旧**，账必须这么做。
- **税务上**：汇算清缴时可以选一次性税前扣除，这是**税会差异**，由我们税务组在年底汇算时做纳税调整搞定。

所以"能一次性税前扣"是年底税务上的事，**不影响你这台设备先入固定资产**。别拿"反正能一次性扣"当借口把设备拆成费用报。

#### 10.3 拆分采购——一套东西合计达标，就按一项固定资产

正文 2.5 说了别拆票规避固定资产标准，这里给个更明确的判据：

> **同一供应商、同一批次、功能上成一套的物品，合计金额达到 5,000 元固定资产标准的，按一项固定资产管理**，不管你开了几张票。

真实案例：某同事买工作站，主机、显示器、扩展坞分了三张票，每张都是几千块，单看都不到 5,000，合计 1.2 万。他想按办公费一次性报掉。**不行**——这三样功能上就是一台工作站，合计 1.2 万，必须按一项固定资产建卡折旧。系统加上票通和金蝶的比对，同供应商、同批次、连号的票，我们一眼就看出来了。

#### 10.4 外购软件——单独定价、用超 1 年、达标的入无形资产

软件分两种处理：

- **外购的软件，单独定价、使用年限超过 1 年、金额达到 5,000 元标准**的：作为**无形资产**入账，分期摊销。比如买断式的一套设计软件、一套企业级系统授权。
- **年度/月度的 SaaS 订阅**（按期付费、用完即止）：**费用化**，正文 2.14 讲过，年付的还要按月待摊（填服务起止期间）。

判断口诀：**买断、长期用、金额大 → 无形资产摊销；按期订阅、用完即止 → 费用化（年付的待摊）。**

#### 10.5 装修、大修、改良——达标的资本化，别一笔进当期

租来的办公室搞了一次大装修，或者设备做了大修改良，这种**金额大、受益期长**的，不能一笔全进当期费用：

- **租入办公室的装修改良**：达到标准的，进**长期待摊费用**，按租赁期（或受益期）分期摊销。
- **自有固定资产的大修理、改良**：达到资本化标准（延长使用寿命、提升性能）的，并入固定资产价值或单独资本化，不能当期一次费用化。
- **日常的小修小补、维护**：金额小、不改变性能的，当期费用化（管理费用—修理费）。

判断口诀：**让东西"用得更久、性能更强"的大支出，资本化慢慢摊；只是"维持原状"的小维护，当期报掉。** 大额装修、大修的报销，提前找核算组，别等装完了拿张几十万的发票来一次性报。

### 第十一章 费用会计初审到底在看什么（SOP 显性化 + 必备附件对照表）

总有同事问"你们财务初审到底看啥，我提前自己对一遍能少退点单吗？"——能。这一节把我们费用会计拿到一张单子之后的动作顺序摆出来，再给一张各科目必备附件的对照表，你提交前照着自检。

#### 11.1 费用会计初审的动作顺序

我们拿到单子，大致按这个顺序过：

1. **四个一致**——这是第一关，任何一个对不上直接退：
   - 发票**抬头主体** = 报销人所属的法人主体（别拿 A 子公司抬头的票在 B 主体报，见第十二章）；
   - 发票**金额** = 明细行金额 = 合计金额（三个数对得上）；
   - 发票**日期** ∈ 费用所属期（没过认证期、没跨年超期）；
   - 发票**品名** ↔ 所选科目的性质相符（开"餐饮"的别选"办公费"）。
2. **看发票状态**——在票通已验真、在认证期内、无重复（没被报过）、无红冲作废记录。
3. **看连号、同店扎堆**——同一商户连号票、同部门短期反复在同一家开票，飘黄重点看。
4. **看金额、品名合理性**——金额是不是卡在审批阈值正下方（采购 2,800/2,900、招待 780/790），品名和事由对不对得上。
5. **看附件完整性**——按下面那张对照表，该有的附件齐不齐。

前四步任何一步不过，基本就退了。第五步附件不齐的，看情况退或要求补。

#### 11.2 各科目必备附件对照表（提交前照着自检）

| 科目 | 必备附件 |
|---|---|
| 差旅—大交通（机票/高铁） | 行程单/电子客票 + 出差申请单 |
| 差旅—住宿 | 住宿发票 + 出差申请单，且住宿天数与出差天数一致 |
| 差旅—市内交通 | 网约车行程单；单笔 > 100 元附行程明细说明 |
| 业务招待费 | 招待对象单位/姓名/人数 + 我方陪同人数；单次 ≥ 3,000 元附接待安排或来访邮件 |
| 办公用品 | 发票 + 销货清单（无清单不收） |
| 设备采购 | 采购合同/订单号；达固定资产标准的附验收单 |
| 研发费用 | 研发项目号；委外测试附合同；样机试制附领料单/试制记录 |
| 会议费 | 会议通知/议程、签到表、纪要至少其中两项 |
| 培训费 | 报名记录 + HR 岗位相关性确认；考证类附证书/成绩 |
| 劳务费（个人） | 劳务协议 + 成果证明 + 个人身份信息 + 个税代扣记录 |
| 福利费/团建费 | 发放签收名单/参加人员名单 + 活动说明 |
| 咨询费 | 咨询合同 + 书面成果（报告/方案/意见书） |
| 中介服务费 | 居间/代理协议，写明撮合事项 |
| 市场推广费 | 推广合同/方案 + 用途说明；带广告露出的附露出条款 |

### 第十二章 票据风险进阶与"一票否决"边界

正文 1.2 和第六章讲了发票硬条件和高频踩坑，这一年又出了几种更隐蔽的票据问题，专门补，并且把"一票否决（无论金额都必退）"和"可凭说明放行"的边界划清楚。

#### 12.1 几种进阶的问题票

- **红冲/作废票**：票面看着完全正常，但销货方已经红冲或作废了，票通能查到红冲记录。**自己提交前先在票通查一遍状态**，别等被退。
- **阴阳金额票**：发票开 580、实际只付了 480 这种。所以正文里要求**单笔 1,000 元以上对私报销随附支付成功凭证截图**，金额对不上的直接退。
- **代开票（虚开嫌疑）**：几个信号——品名跟你岗位/事由完全无关（一个做研发的报"建材"）、销货方经营范围跟开的票对不上（卖五金的开"咨询费"）、短期内同一个销货方反复给不同员工开票。碰到这种我们会重点核查，必要时移交。
- **个人抬头票**：抬头开成员工个人名字的。**机理上：个人抬头不能抵进项，而且公司给员工报个人抬头的消费，本质是变相发了笔不缴个税的钱，是稽查重点。** 所以个人抬头票硬退，没商量。
- **跨主体票**：A 主体抬头的票拿到 B 主体来报。**两头都是问题**：A 这边凭空多了张没有对应业务的票，B 这边抵不了进项也入不了账。必退重开。

#### 12.2 一票否决（无论金额，一律退）

下面这些情形，**不管金额多小，一律退，没有"通融"一说**：

1. 假票、查无此票、红冲票、作废票、重复报销的票；
2. 个人抬头票、跨主体抬头票；
3. 该代扣个税而未代扣的（个人劳务）；
4. 发票超期（超报销时限、过认证期，以制度时限为准）；
5. 设备拆票规避固定资产标准的。

#### 12.3 可凭说明 + 升级放行（不是一票否决，但要走流程）

下面这些不是硬伤，可以凭合理说明 + 升一级审批放行：

1. **紧急情况未事前审批**：事后双签 + 3 个工作日内补单（时限以制度为准）；
2. **小额发票遗失**：从严，凭其他佐证 + 说明，金额大的不通融；
3. **招待人均小幅超标**：写明原因，升级审批；
4. **事前额度小幅超支**（一般掌握在 10% 以内）：写明原因，升级审批。

**一句话区分：动了"真实性、合规底线"的（假票、个人抬头、跨主体、该扣税没扣、拆票避固资）一票否决；只是"程序瑕疵、小幅超标"的，可以说明 + 升级放行。**

### 第十三章 跨科目归类争议怎么判，以及"其他"科目的上限

最后讲讲拿不准科目时的判定流程，还有大家最爱用的那个万能筐——"其他"科目，今年要给它套上笼子。

#### 13.1 跨科目归类争议的判定流程

碰到一笔费用，几个科目都像、又都不完全像，按这个顺序判：

1. **看实质，不看发票上写什么**——对方开"咨询费"不代表就是咨询费，看你实际买的是什么（第七章那些判据）。
2. **看主要用途归主科目**——一笔费用如果横跨几个科目，按"主要是为了干什么"归到主科目，别硬拆得太碎。比如一场带了点餐饮的客户洽谈会，主体是会议就归会议费，餐饮单独开的那部分按招待拆出来。
3. **看税务后果择重**——如果两个科目都说得通，但一个能抵/能扣、一个不能，**不要为了多抵多扣就往有利的科目靠**，按实质来。被查出来"为了抵扣而错配科目"，比单纯记错科目性质严重得多。
4. **拿不准的，走"财务答疑"或直接问对接的费用会计**，别自己拍脑袋选个差不多的就提交。大额的、敏感的（咨询、赞助、研发、视同销售、资本化），**事前**找税务组。

#### 13.2 "其他"科目的上限——别什么都往筐里扔

"管理费用—其他""销售费用—其他"这种筐，是给真的归不进任何明确科目的零星支出留的，不是给你偷懒用的。今年定两条硬规矩：

1. **单笔 ≥ 2,000 元想进"其他"科目的，必须经财务复核确认 + 写明为什么不能归到某个明确科目。** 说不清的，退回去重新归类。
2. **某个部门的"其他"科目占该部门总费用的比例超过 5% 的，列入内控关注名单**，季度费控分析会点名，要求把"其他"里的费用拆解归位。

为什么管这么严？因为"其他"科目是审计和税务最爱翻的地方——一堆说不清性质的支出堆在"其他"里，既影响费用分析的准确性，又容易藏着不合规的东西。**一笔费用，只要能说清它是干什么的，就一定有对应的明确科目；说不清，才是真问题。** 所以请大家报销前先想清楚性质（这也是整本手册开头那句话），实在归不进去的零星小额，才用"其他"，而且要写清楚理由。
