# 企业人事管理 — 系统与数据实体字典

**文档编号:** HRM-SYS-003
**版本:** 3.1
**主责部门:** 人力资源信息化 (HRIS) & 数据工程
**生效日期:** 2026-01-01
**关联文档:** `01-业务流程SOP.md` (HRM-SOP-001)、`02-业务规则与制度.md` (HRM-POL-002)、`04-事件与工作流编排.md` (HRM-WF-004)。

本文档是企业人事管理技术栈的权威数据契约。此处的实体(对象 Objects)与属性命名即为 SOP 与规则文档引用的规范名称。

---

## 1. 记录系统 (System of Record)

| 系统 | 角色定位 | 主数据归属(系统对象) |
|---|---|---|
| **核心人力系统 (HRIS)** | 核心人事 | `员工`、`岗位`、`部门`、`组织架构`、`劳动合同`、`薪资结构`(主数据) |
| **招聘入职系统 (Onboarding系统)** | 入职管理 | `入职单`、`Offer`、`入职材料`、`报到记录` |
| **绩效管理系统 (绩效系统)** | 绩效管理 | `考核周期`、`绩效目标`、`评估`、`绩效结果`、`绩效改进计划` |
| **薪酬福利系统 (薪酬系统)** | 薪酬福利 | `调薪单`、`奖金`、`发薪批次`、`社保公积金`、`福利` |
| **审批工作流引擎 (审批引擎)** | 申请审批 | `申请单`、`审批流`、`离职单`、`交接单` |
| **考勤系统 (考勤系统)** | 考勤工时 | `任务`、`工时`、`假期额度`、`考勤异常`、`奖惩记录` |
| **IT账号与资产系统 (IT资产系统)** | 账号资产 | `账号开通申请`、`域账号`、`办公资产领用单` |
| **法务合规系统 (法务系统)** | 合规协议 | `离职证明`、`竞业协议`、`保密协议` |

## 3. 系统间集成与事件交换

| 源系统 → 目标系统 | 交换的事件 | 载荷主键 |
|---|---|---|
| 招聘入职系统 → 核心人力系统 | `劳动合同已签署` / `员工档案已建立` | `employee_id`, `contract_id` |
| 招聘入职系统 → IT资产系统 | `账号开通已申请` | `provision_id`, `employee_id` |
| IT资产系统 → 招聘入职系统 | `账号已开通` | `provision_id`, `domain_account` |
| 绩效系统 → 薪酬系统 | `绩效结果已发布` | `result_id`, `grade`, `bonus_factor` |
| 薪酬系统 → 考勤系统 | (拉取) 工时与考勤异常用于计薪 | `employee_id`, `pay_month` |
| 考勤系统 → 薪酬系统 | `考勤扣款已生成` | `employee_id`, `pay_month`, `deduction` |
| 审批引擎 → 核心人力系统 | `申请已批准`(调岗/晋升回写岗位) | `request_id`, `employee_id`, `position_id` |
| 审批引擎 → 薪酬系统 | `调薪已核准` | `adjustment_id`, `new_salary` |
| 审批引擎 → IT资产系统 | `交接已完成`(触发回收) | `handover_id`, `employee_id` |
| 薪酬系统 → 法务系统 | `社保已停缴` | `social_id`, `employee_id` |
