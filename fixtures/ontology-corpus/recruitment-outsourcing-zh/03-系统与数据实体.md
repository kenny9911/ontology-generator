**文档编号:** RAAS-SYS-003
**版本:** 3.1
**归属部门:** 招聘技术平台组 (Recruitment Tech Platform) 与数据工程
**生效日期:** 2026-01-01
**关联文档:** `01-业务流程SOP.md` (RAAS-SOP-001)、`02-业务规则与制度.md` (RAAS-POL-002)、`04-事件与工作流编排.md` (RAAS-EVT-004)。

---

## 1. 记录系统 (Systems of Record)

| 系统 | 角色 | 作为记录系统持有 (System of Record for) |
|---|---|---|
| **TalentFlow ATS** | 招聘业务核心 | `RecruitmentRequest`、`Position`、`JobDescription`、`Candidate`、`Resume`、`Application`、`ScreeningCall`、`Interview`、`AssessmentReport`、`RecommendationPackage`、`Offer` |
| **MatchEngine** | 简历解析与匹配 | `MatchScore`、解析后的 `Skill` 标签 |
| **ClientPortal** | 客户协作门户 | 客户反馈、JD 审批、推荐包查看、Offer 审批 (视图层, 不持有主记录) |
| **VerifyHub** | 背景调查 | `BackgroundCheck` |
| **ContractBilling** | 合同与计费 | `ServiceContract`、`ServiceFee`、`BillingMilestone`、`GuaranteePeriod` |
| **CommChannel** | 通知网关 | 短信 / 邮件 / 外呼通知日志 (无业务主记录) |

## 3. 系统集成 (Integrations)

| 来源系统 | 目标系统 | 集成方式 | 交换内容 |
|---|---|---|---|
| TalentFlow ATS | MatchEngine | 异步 API + 事件 | 简历文本与职位要求 → 匹配评分 (`MatchScore`) |
| TalentFlow ATS | ClientPortal | REST + Webhook | 推荐包、JD、Offer 推送; 客户反馈回流 |
| TalentFlow ATS | VerifyHub | REST API | Offer 触发背调; 背调结论回写 |
| TalentFlow ATS | ContractBilling | 事件总线 | 入职 / 保证期事件 → 触发回款里程碑 |
| TalentFlow ATS | CommChannel | 消息队列 | 邀约 / 提醒 / 通知外发 |
| ContractBilling | ClientPortal | REST | 发票与回款状态展示 |

---

## 4. 系统间事件交换 (Event Exchange)

| 事件 | 生产系统 | 消费系统 |
|---|---|---|
| `简历已采集 (ResumeIngested)` | TalentFlow ATS | MatchEngine |
| `匹配评分已生成 (MatchScored)` | MatchEngine | TalentFlow ATS |
| `推荐包已交付 (RecommendationDelivered)` | TalentFlow ATS | ClientPortal |
| `客户已决策 (ClientDecisionRecorded)` | ClientPortal | TalentFlow ATS |
| `Offer已发出 (OfferIssued)` | TalentFlow ATS | VerifyHub、ContractBilling |
| `背调已完成 (BackgroundCheckCleared)` | VerifyHub | TalentFlow ATS |
| `候选人已入职 (CandidateOnboarded)` | TalentFlow ATS | ContractBilling |
| `回款里程碑已触发 (BillingMilestoneTriggered)` | ContractBilling | ClientPortal |
| `保证期内离职 (GuaranteeBreached)` | ContractBilling | TalentFlow ATS |

事件载荷的字段定义详见 `04-事件与工作流编排.md` 的事件目录。
