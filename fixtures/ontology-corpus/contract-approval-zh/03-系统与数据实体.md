## 1. 业务系统 (记录系统) 总览

| 系统 | 记录的核心实体 | 角色 |
|---|---|---|
| **CLM 合同管理平台** | `合同`、`合同类型`、`合同模板`、`条款`、`审批流`、`审批节点`、`变更单`、`续约` | 合同主记录与审批编排的系统 |
| **法务审查系统** | `法务意见` | 法务审查与意见的记录系统 |
| **财务共享中心系统 (FSSC)** | `付款计划`、`发票` | 税务、回款、信用与开票的记录系统 |
| **风控合规平台 (GRC)** | `风险等级` | 风险评定与合规筛查的记录系统 |
| **用印管理系统** | `用印申请`、`印章` | 用印与盖章的记录系统 |
| **电子签章平台** | (签署存证) | 电子签发起与存证 |
| **主数据系统 (MDM)** | `相对方` | 客户/供应商/员工主数据与信用档案 |
| **电子档案系统 (DMS)** | (归档台账) | 合同正本归档与版本留存 |

## 3. 系统间集成

| 集成 | 方向 | 内容 |
|---|---|---|
| CLM ↔ 主数据系统 (MDM) | CLM 调用 MDM | `校验相对方` 拉取相对方主数据与信用档案 |
| CLM → 法务审查系统 | CLM 推送 | 提交审查时推送合同与条款, 回收 `法务意见` |
| CLM → 财务共享中心系统 (FSSC) | CLM 推送 | 提交审查时推送金额/税率, FSSC 回写 `付款计划` 与财务结论 |
| CLM → 风控合规平台 (GRC) | CLM 推送 | 提交审查时推送合同要素, GRC 回写 `风险等级` |
| 用印管理系统 ↔ 电子签章平台 | 双向 | 电子签盖章调用与签署存证回传 |
| CLM → 电子档案系统 (DMS) | CLM 推送 | 盖章后推送合同正本归档, DMS 回写归档台账号 |
| FSSC → CLM | FSSC 回写 | 发票开具与付款状态回写履约记录 |

## 4. 系统间交换的事件 (摘要)

| 事件 | 生产系统 | 消费系统 |
|---|---|---|
| `合同已提交审查` | CLM 合同管理平台 | 法务审查系统、FSSC、风控合规平台 |
| `法务审查完成` | 法务审查系统 | CLM 合同管理平台 |
| `财务审查完成` | 财务共享中心系统 (FSSC) | CLM 合同管理平台 |
| `风险等级已评定` | 风控合规平台 (GRC) | CLM 合同管理平台 |
| `合同审批通过` | CLM 合同管理平台 | 用印管理系统 |
| `合同已盖章` | 用印管理系统 / 电子签章平台 | 电子档案系统 (DMS) |
| `合同已归档` | 电子档案系统 (DMS) | CLM 合同管理平台 |
| `发票已开具` | 财务共享中心系统 (FSSC) | CLM 合同管理平台 |

> 完整事件目录与载荷字段见 `04-事件与工作流编排.md`。
